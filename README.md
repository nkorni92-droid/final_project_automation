## Примечания по запуску

### Важно!
1. **Сайт Читай-город использует защиту Cloudflare**, поэтому API запросы могут возвращать 403 ошибку. Тесты адаптированы для работы с этой особенностью.
2. **Для UI тестов требуется ChromeDriver**. Проект использует webdriver-manager для автоматической загрузки.
3. **Регистрация не требуется** - тесты работают с публичными страницами сайта.

### Решение проблем

#### Ошибка WinError 193
Эта ошибка возникает, если ChromeDriver несовместим с системой.

**Решение:**
1. Скачайте ChromeDriver вручную с https://chromedriver.chromium.org/
2. Поместите chromedriver.exe в папку с проектом
3. Или добавьте путь к chromedriver в переменную PATH

#### 403 ошибка в API тестах
Это ожидаемое поведение из-за защиты Cloudflare. Тесты помечаются как пропущенные с соответствующим сообщением.

### Запуск тестов

```bash
# Установка зависимостей
pip install -r requirements.txt

# Все тесты
pytest tests/ -v

# Только UI тесты
pytest tests/ -v -m ui

# Только API тесты
pytest tests/ -v -m api

# С Allure отчетами
pytest tests/ -v --allure

## Описание проекта

Проект автоматизации тестирования интернет-магазина "Читай-город" на основе финальной работы по ручному тестированию. Включает UI и API тесты.
